# Contributing to Aesthetic Evaluation Research

Thank you for your interest in this research! This project investigates critic-audience rating disparities through cognitive psychology and aesthetic evaluation theory.

## How to Contribute

### 1. Report Issues
- Found an error in the analysis or theory application?
- Noticed a reference citation issue?
- Have suggestions for clarification?

Please open an issue with:
- Clear description of the problem
- Specific line numbers or sections if applicable
- Suggested correction or improvement

### 2. Suggest Enhancements
- Additional theoretical frameworks to consider?
- Alternative statistical approaches?
- Expansion to other documentaries or media?

Create an issue labeled `enhancement` with your ideas.

### 3. Improve Documentation
- Clearer explanations of methods?
- Additional examples or visualizations?
- Better formatting?

Submit a pull request with your improvements.

### 4. Extend the Research
- Want to replicate this analysis on different films?
- Testing alternative theoretical frameworks?
- Cross-cultural validation?

Please open an issue first to discuss your approach!

## Technical Setup

### Requirements
- XeLaTeX (TeX Live or MacTeX)
- Git

### Building the PDF
```bash
bash scripts/build.sh
```

Output: `output/whitepaper.pdf`

### File Structure
- `source/whitepaper.tex` - Main document
- `scripts/build.sh` - Build script
- `output/` - Compiled PDF

## Ethical Considerations

This work was authored by an LLM. Contributors should:
1. Verify theoretical claims independently
2. Check all reference citations
3. Note any speculative connections between theories
4. Clearly distinguish empirical findings from theoretical interpretations
